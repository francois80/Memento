HTML Exercice 1 : Créer une structure html5 valide.


      <!DOCTYPE html >
      <html lang="fr" dir="ltr">
      <head>
        <meta charset="utf-8">
        <link rel="stylesheet" href="/assets/css/style.css">
        <title>Création HTML et CSS</title>
      </head>
      <body>
        <p> Exercice 1 : Créer une structure html5 valide.</p>
      </body>
      </html>

Pour plus de lisibilité, il est préférable d'indenter son code. Ce qui donne la représentation ci-dessous : 

      
      <!DOCTYPE html>
      <html lang="fr" dir="ltr">
        <head>
          <meta charset="utf-8">
          <link rel="stylesheet" href="/assets/css/style.css">
          <title>Création HTML et CSS</title>
        </head>
      <body>
        <p> Exercice 1 : Créer une structure html5 valide.</p>
